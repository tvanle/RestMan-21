🚀 CHẠY NHANH - 3 BƯỚC

┌──────────────────────────────────────────────────────────────┐
│  BƯỚC 1: CÀI ĐẶT CÔNG CỤ                                    │
└──────────────────────────────────────────────────────────────┘

    ✅ JDK 8 or 11
       → https://adoptium.net/

    ✅ MySQL 8.0
       → https://dev.mysql.com/downloads/mysql/

    ✅ Apache Tomcat 9
       → https://tomcat.apache.org/download-90.cgi
       → Giải nén vào: C:\apache-tomcat-9.0.80

    ✅ VSCode
       → https://code.visualstudio.com/

┌──────────────────────────────────────────────────────────────┐
│  BƯỚC 2: SETUP PROJECT                                       │
└──────────────────────────────────────────────────────────────┘

    Windows: Double-click setup.bat

    Hoặc mở Terminal và chạy:

    1. Setup database:
       mysql -u root -p < database\restman.sql

    2. Download MySQL Connector:
       https://dev.mysql.com/downloads/connector/j/
       Copy mysql-connector-java-8.x.jar vào WebContent\WEB-INF\lib\

    3. Cấu hình DB password trong src\dao\DAO.java

┌──────────────────────────────────────────────────────────────┐
│  BƯỚC 3: BUILD VÀ CHẠY                                       │
└──────────────────────────────────────────────────────────────┘

    Windows:

    build.bat     → Build project
    run.bat       → Chạy Tomcat

    Hoặc trong VSCode:

    1. Cài Extensions: Extension Pack for Java, Tomcat for Java
    2. Ctrl+Shift+P → Tomcat: Add Tomcat Server
    3. TOMCAT SERVERS → Add Package → target\RestMan.war
    4. Right-click server → Start

    Browser: http://localhost:8080/RestMan/

🐛 TROUBLESHOOTING

    ❌ Cannot connect to database
       → Kiểm tra MySQL đang chạy: net start MySQL80
       → Check password trong src\dao\DAO.java

    ❌ 404 Not Found
       → URL phải là: http://localhost:8080/RestMan/
       → Check WAR deployed: dir %TOMCAT%\webapps\

    ❌ Port 8080 in use
       → netstat -ano | findstr :8080
       → taskkill /PID <PID> /F

    ❌ Compilation failed
       → Check JAVA_HOME: echo %JAVA_HOME%
       → Ensure mysql-connector.jar in WebContent\WEB-INF\lib\
