🚀 BƯỚC TIẾP THEO:

┌──────────────────────────────────────────────────────────────┐
│  OPTION 1: Chạy với Tomcat trên VSCode                      │
└──────────────────────────────────────────────────────────────┘

   1. Cài Extension "Tomcat for Java" trong VSCode

   2. Thêm Tomcat server:
      Ctrl+Shift+P → Tomcat: Add Tomcat Server
      → Chọn thư mục Tomcat (C:\apache-tomcat-9.0.xx)

   3. Deploy WAR:
      TOMCAT SERVERS (sidebar)
      → Right-click server
      → Add Package
      → Chọn: target\RestMan.war

   4. Start server:
      Right-click server → Start

   5. Mở browser:
      http://localhost:8080/RestMan/

┌──────────────────────────────────────────────────────────────┐
│  OPTION 2: Chạy với script (Windows)                        │
└──────────────────────────────────────────────────────────────┘

   Double-click: run.bat

   Hoặc terminal:

   run.bat

   → Browser: http://localhost:8080/RestMan/

┌──────────────────────────────────────────────────────────────┐
│  OPTION 3: Deploy thủ công                                  │
└──────────────────────────────────────────────────────────────┘

   1. Copy WAR file:
      copy target\RestMan.war C:\apache-tomcat-9.0.xx\webapps\

   2. Start Tomcat:
      C:\apache-tomcat-9.0.xx\bin\startup.bat

   3. Check logs:
      C:\apache-tomcat-9.0.xx\logs\catalina.out

   4. Access:
      http://localhost:8080/RestMan/

⚠️  TRƯỚC KHI CHẠY, PHẢI:

   1. ✅ Cài MySQL và khởi động service
      net start MySQL80

   2. ✅ Tạo database:
      mysql -u root -p < database\restman.sql

   3. ✅ Download MySQL Connector JAR:
      https://dev.mysql.com/downloads/connector/j/
      → Copy vào: WebContent\WEB-INF\lib\

   4. ✅ Cấu hình password trong src\dao\DAO.java:
      private static final String DB_PASSWORD = "your_password";

   5. ✅ Rebuild nếu thay đổi code:
      mvn clean package
